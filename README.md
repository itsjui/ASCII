#ModernShop

ModernShop é uma landing page moderna e responsiva para uma loja virtual fictícia, desenvolvida com **HTML**, **CSS** e **JavaScript**.  
O objetivo do projeto é apresentar produtos de forma atrativa, com animações suaves, design limpo e foco em usabilidade.

---
##Funcionalidades

- Barra de navegação com links interativos  
- Seção *hero* com destaque para coleções  
- Cards de produtos com animação de entrada  
- Etiquetas de desconto geradas dinamicamente  
- Rodapé com informações da loja e redes sociais  
- Layout totalmente responsivo
